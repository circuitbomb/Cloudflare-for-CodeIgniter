Cloudflare-for-CodeIgniter
==========================

Just a small client api library I put together for accessing and modifying settings in Cloudflare from CodeIgniter

Does not implement (yet):
	-A NAME record updating
	-DNS Record Deletion
	-DNS Record Creation
